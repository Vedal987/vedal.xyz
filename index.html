Falcon Heavy in Unity

Falcon Heavy in Unity

I attempted to recreate the SpaceX Falcon Heavy launch in Unity. The rockets are fully automated and use simulated thrust vectors that then account for many different scenarios in order to get the perfect landing, including their fuel, xyz velocity and their rotation.

VR Minecraft

Tic Tac Toe Machine Learning

Tic Tac Toe Machine Learning

Using Python I created a decision tree based Artificial Intelligence that learns by itself, with about 10 minutes of training it becomes unbeatable meaning it is only possible to lose or draw to this bot as it makes perfect decisions. These decisions are based on thousands of games that it has played with itself in order to slowly improve.

Pac-Man Machine Learning

Pac-Man Machine Learning

Using Unity and C# I created a bot that learns to play a 3D version of Pac-Man that I created. It was inspired by this video and after several hours of training it became better than me at the game.

Connect 4 Machine Learning

Connect 4 Machine Learning

Using Python I created a decision tree based Artificial Intelligence that learns by itself to play Connect 4. I left this training for several days until the model size was several gigabytes big, this was due to the vast number of possible outcomes in Connect 4. By this time the bot had become moderately good at Connect 4 but was not perfect as it had not been training for long enough. The entire event was streamed at Twitch for viewers to watch.

Alice

Alice

Alice is a voice controlled AI that I created using Python, a Rasperry Pi and some programmable LED lights. The AI is always listening and wakes when hearing "Alice" to respond to many commands that mostly control the lighting in my room. This means I can say "Alice lights rainbow" for a cool RGB rainbow theme in my room. The bot also interacts with my computer through a web server so that I can send commands from a python terminal on my computer or Alice can control specific elements on my computer like applications open or turn it off.

Twitch Plays Dragon Quest IX

Twitch Plays Dragon Quest IX

Inspired by the likes of TwitchPlaysPokemon I created a python program that listens to my twitch chat and interprets commands like "A" or "Right 5" in order to play Dragon Quest IX through the desmume emulator. The twitch chat ended up surprisingly getting quite far into the game and it was cool to see a small community work together like that.

donttap.com Bot

donttap.com Bot

After being challenged by my friend to beat him at donttap.com and failing substantially I loaded up Python and set up a bot that automatically detected where the black tiles were on screen in order to click them. After a lot of optimisation I ended up reaching a high score of 2511 which was by far the highest I had seen any body (or bot) get on the game and definitely my friend.
